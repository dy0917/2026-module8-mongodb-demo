Docker files for DB demo
#How to use
1. Make sure DockerDesk is running
2. Use the Docker command to run the yml file. Ex: "docker compose -f ./redis.yml up -d"

MongoDB exercise
1. login the Docker container terminal(click the triple dot)
2. use the command to login mongoDB \\ mongosh -u "dbadmin" -p "Password1"

Mysql exercise
1. Install MySQL Workbench
2. Create connection with details in mysql yml file

Redis exercise
1. login the Docker container terminal(click the triple dot)
2. use the command to login redis \\ redis-cli -a YourSecurePassword
